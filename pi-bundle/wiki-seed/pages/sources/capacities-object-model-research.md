# Capacities Object Model Research

## Source details

This note summarizes the research used to design the new wiki around objects, properties, relationships, dashboards, and generated views.

## Key takeaways

- object-based thinking maps well to Markdown notes with strong frontmatter
- object properties become structured fields
- relations become typed ID arrays
- dashboards can be normal Markdown notes
- nixpi-wiki can maintain summaries, links, and indexes over time
